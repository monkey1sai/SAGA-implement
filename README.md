# 人體救助行為偵測系統 (Human Distress Detection)

此專案旨在利用姿態辨識技術偵測人體在不同情境下的求救行為，提供即時預警與援助。

## 求救情境說明

本系統針對以下三種關鍵求救情境進行偵測：

### 1. 雙手舉起揮手救助 (Waving for Help)
*   **情境描述**：當人類雙手高舉過頭並進行左右揮動時，系統將判定為強烈的救助訊號。
*   **偵測重點**：雙手手腕關節位置高於頭部，且具備規律性的水平位移特徵。

### 2. 行進間或站立跌倒 (Fall Detection)
*   **情境描述**：偵測人類在站立或走路過程中突然發生的跌倒行為。
*   **偵測重點**：身體重心高度（Hip/Torso）在短時間內劇烈下降，且最終呈現水平或蜷縮姿勢。

### 3. 呼吸道阻塞 (Choking - Heimlich Maneuver)
*   **情境描述**：偵測人類因呼吸道阻塞而表現出的特徵動作（如：雙手掐脖子），判斷是否需要進行哈姆立克急救法。
*   **偵測重點**：雙手靠近頸部區域，且伴隨身體前傾或焦慮不安的姿態特徵。
